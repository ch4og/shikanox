* Maak file

This repository uses [[https://codeberg.org/jjba23/maak][maak]].

Instead of running long guix invocations manually, you can for example do:

#+BEGIN_SRC sh
  maak fmt
  maak update
#+END_SRC

(Hacky maak wrapper binary loaded to =$PATH= with direnv)
